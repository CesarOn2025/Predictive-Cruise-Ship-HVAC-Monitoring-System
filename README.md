# Predictive Cruise Ship HVAC Monitoring System

## Project Overview

This project demonstrates a conceptual MVP (Minimum Viable Product) for predictive HVAC monitoring in cruise ship operations using:

- Python
- APIs
- Cloud Computing Concepts
- NLP (Natural Language Processing)
- Predictive Analytics
- Dashboard Visualization
- Microsoft Azure Concepts

The MVP simulates operational HVAC sensor data to demonstrate how predictive maintenance workflows could support proactive cruise ship HVAC operations.

# Important Clarification

This project represents a conceptual MVP and architecture simulation developed as part of AI studies at Houston Community College.

The current implementation uses simulated operational HVAC data and would require real-world integration with HVAC infrastructure, sensors, BMS systems, APIs, and cloud services for production deployment.
